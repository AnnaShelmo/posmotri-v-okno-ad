# Проект "Посмотри в окно"

## Описание
Простой веб-проект, позволяющий пользователю выбрать город и время суток, после чего отображается список видео из выбранного города. Проект сделан с использованием HTML, CSS и JavaScript.

## Структура проекта

posmotri-v-okno-ad
├── .GitHub
├── fonts
│ ├── fonts.css
│ └── 6 файлов шрифтов
├── scripts
│ └── script.js
├── styles
│ ├── styles.css # Основные стили проекта
│ ├── preloader.css # Стили прелоадера
│ └── error.css # Стили ошибки
└── index.html # Главная страница


### Папки и файлы
- **fonts/** – содержит файлы шрифтов и CSS для подключения шрифтов.
- **scripts/** – содержит JavaScript-код проекта.
- **styles/** – содержит CSS-файлы:  
  - `styles.css` – основные стили страницы и компонентов, форма поиска, карточки видео, кнопки и пр.  
  - `preloader.css` – стили прелоадера (анимация загрузки).  
  - `error.css` – стили отображения ошибки.
- **index.html** – главная страница с разметкой: форма поиска, контейнер для видео, карточки видео, шаблоны для прелоадера и ошибок.

## Особенности проекта
- Форма поиска с кастомными чекбоксами.
- Адаптивный список видео с прокруткой.
- Прелоадер для загрузки видео.
- Обработка ошибок при отсутствии видео для выбранного города.
